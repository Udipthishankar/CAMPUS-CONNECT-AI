Campus Connect AI

Competition: Frontend Web Development Competition 2026

1. Project Overview

Campus Connect AI is a web-based conversational interface designed to provide students with 24/7 instant access to academic and administrative information using Generative AI.

2. Features

  • Generative AI Chat Interface
  • Natural Language Processing (NLP)
  • Context-Aware Conversations (RAG)
  • Secure Student/Admin Login
  • Mobile-Responsive Design
  • Admin Dashboard for Knowledge Management

3. Technology Stack

Frontend: React.js, Tailwind CSS

Backend/AI: Node.js, Python, Flask, LangChain, Google Gemini API

Deployment: Vercel
